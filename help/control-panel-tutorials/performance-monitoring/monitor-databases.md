# Monitorar bancos de dados

Todos os ativos, workflows e dados da instância do Campaign são armazenados no banco de dados. O banco de dados que atinge sua capacidade máxima pode levar a problemas com sua instância (incapacidade de fazer logon, enviar emails etc.). Portanto, é essencial monitorar regularmente os bancos de dados das instâncias para garantir o melhor desempenho.

Saiba como monitorar o uso do banco de dados de suas instâncias.
